## File system & Terminal

---

1. 컴퓨터 OS
    * Operating System, 운영체제
        > Windows , OS X
    
    * File System
        > 트리구조 저장 체계 (폴더, 파일, 확장자)
    
    * 절대경로와 상대경로
        - 절대 경로 : c:\user\docs\...

        - 상대 경로 : ../../현재폴더
    * Terminal(Console, CMD)
        - Graphical User Interface : 일반적인 화면

        - Command Line Interface : 마우스가 아닌 키보드로 명령을 입력 프로그램 실행
            >mkdir 폴더명 : make directory  
            >cd : change directory
            >dir ..\현재폴더 : 현재폴더로 가기
